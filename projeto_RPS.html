RPS Game

GÉNERO Jogo
Linguagens Utilizadas HTML | CSS | JAVASCRIPT
ANO 2025

Rock-Paper-Scissors é um jogo interativo desenvolvido em HTML, CSS e JavaScript, criado com o objetivo de reforçar competências fundamentais em desenvolvimento web. O foco principal deste projeto foi a aplicação de conceitos de design modernoe a sincronização de animações visuais com a lógica do jogo.
